# 💰 Expense Tracker Dashboard

## 📌 Deskripsi
Expense Tracker Dashboard adalah aplikasi berbasis web untuk mencatat dan memvisualisasikan pengeluaran secara interaktif.  
Aplikasi ini membantu pengguna memahami pola pengeluaran melalui list transaksi, pie chart, dan ringkasan bulanan.

---

## 🚀 Fitur Utama
- ➕ Input transaksi (nama, jumlah, kategori, tanggal)
- 📋 List transaksi dengan filter dan sorting
- 📊 Visualisasi pengeluaran menggunakan pie chart
- 💵 Total keseluruhan saldo transaksi

---

## ⚙️ Fitur Tambahan
- 🏷️ Menambahkan kategori custom
- 📅 Monthly summary (ringkasan per bulan)
- 🔽 Sorting transaksi (ascending, descending, default)
- 🚨 Highlight pengeluaran di atas batas tertentu
- 🌙 Dark / Light mode toggle
- 🔔 Notifikasi interaktif menggunakan SweetAlert2

---

## 🛠️ Teknologi yang Digunakan
- HTML
- Tailwind CSS
- JavaScript (Vanilla JS)
- Chart.js
- SweetAlert2
- LocalStorage (untuk penyimpanan data)

---

## 📊 Cara Menggunakan
1. Tambahkan transaksi melalui form
2. Pilih kategori atau tambahkan kategori baru
3. Gunakan filter untuk melihat data tertentu
4. Klik monthly summary untuk melihat detail per bulan
5. Lihat distribusi pengeluaran melalui pie chart
